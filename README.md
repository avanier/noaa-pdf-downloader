# noaa-pdf-downloader

This Ruby script downloads the latest revision of very PDF nautical chart published by NOAA from their catalog.

## Requirements

Developed on Ruby `2.6.3p62` on Linux.

## Usage

```shell
bundle install
./downloader.rb
```
